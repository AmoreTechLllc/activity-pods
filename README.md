# ActivityPods

Check [these slides](https://docs.google.com/presentation/d/1RCBjy754e2Fn2HREzjHLBXe0qljOD0Y7D8jbEiO7Fb4) for more information about this project !

## Getting started

```
yarn install
yarn run bootstrap
docker-compose up -d fuseki
cd boilerplate
yarn run dev
```

## Tests

```
yarn install
yarn run bootstrap
docker-compose up -d fuseki_test
cd tests
yarn run test
```
